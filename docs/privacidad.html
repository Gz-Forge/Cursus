Política de Privacidad

Última actualización: 21 de abril de 2026

Esta política de privacidad describe cómo Cursus ("la aplicación", "nosotros") recopila, utiliza y protege la información de sus usuarios.

1. Información que recopilamos

  • Datos de cuenta: dirección de correo electrónico y contraseña, utilizados para autenticación mediante Supabase.
  • Datos académicos: materias, notas, configuraciones y perfiles que el usuario ingresa voluntariamente en la aplicación.
  • Cámara: la aplicación solicita acceso a la cámara exclusivamente para escanear códigos QR. No se capturan ni almacenan imágenes.

2. Cómo usamos la información

  • Proveer las funcionalidades de la aplicación.
  • Permitir el inicio de sesión y la gestión de perfiles.
  • Facilitar la sincronización puntual entre dispositivos cuando el usuario la solicita explícitamente.
  • No vendemos ni compartimos datos personales con terceros.

3. Almacenamiento y seguridad

Los datos de cuenta se almacenan de forma segura en Supabase, que utiliza cifrado en tránsito (TLS) y en reposo.

Los datos académicos (materias, notas, configuraciones) se guardan únicamente de forma local en el dispositivo. No existe sincronización en tiempo real ni almacenamiento permanente en la nube de estos datos.

Cuando el usuario inicia manualmente una sincronización entre la app de escritorio y la app móvil, los datos académicos se cargan de forma temporal en el servidor exclusivamente para realizar la transferencia entre dispositivos. Una vez completada la sincronización, esos datos son eliminados del servidor de forma inmediata. Este diseño es la razón por la cual la sincronización no es en tiempo real.

4. Retención de datos

Los datos de cuenta se conservan mientras el usuario mantenga su cuenta activa. Los datos académicos nunca se retienen en el servidor más allá del proceso de sincronización. El usuario puede eliminar su cuenta en cualquier momento contactándonos.

5. Menores de edad

La aplicación no está dirigida a menores de 13 años. No recopilamos intencionalmente datos de menores.

6. Cambios a esta política

Podemos actualizar esta política periódicamente. La fecha de última actualización se indica al inicio del documento.

7. Contacto
